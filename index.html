Ejemplos de CSS

Informacion sobre CSS y ejemplos

¿Qué es y para qué sirve CSS? CSS funciona como complemento a la información que forma parte de un sitio web. Mientras que el código en HTML incluye todos los datos, el código en CSS se encarga de darles formato y presentarlos visualmente a través de un navegador.
Describiremos brevemente las funciones de CSS con ejemplos y dejaremos herramientas para su uso.
